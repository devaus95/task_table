# Table Editor 项目验收清单

## 功能验收

### AC 1: 表格显示
- [ ] 打开页面时显示空表格，包含Index、Name、Data Type、Default Value、Comment列
- [ ] Index列是只读的且自动生成
- [ ] 显示"Add Row"和"Delete Row"按钮

### AC 2: 添加变量行
- [ ] 点击"Add Row"按钮后在表格末尾添加新行
- [ ] 新行的所有字段（name, data type, default value, comment）默认为空
- [ ] Index自动生成为当前最大index + 1

### AC 3: 删除变量行
- [ ] 选中行并点击"Delete Row"按钮后，该行被删除
- [ ] 删除后，后续行的Index自动重新计算

### AC 4: 编辑变量名称
- [ ] 输入空名称时，恢复原始值并显示错误提示
- [ ] 输入已存在的名称（不区分大小写）时，显示"Name already exists"错误
- [ ] 输入唯一且非空的名称时，成功保存

### AC 5: 选择数据类型
- [ ] 双击数据类型单元格显示BOOL和INT下拉选项
- [ ] 选择数据类型后，单元格显示所选类型
- [ ] 切换数据类型时，默认值自动更新（BOOL -> TRUE, INT -> 0）

### AC 6: 编辑BOOL默认值
- [ ] 接受输入：true, false, TRUE, FALSE（不区分大小写）
- [ ] 统一显示为大写：TRUE或FALSE
- [ ] 输入其他值时显示错误提示

### AC 7: 编辑INT默认值
- [ ] 接受整数输入，范围：-2147483648到2147483647
- [ ] 输入非整数时显示错误提示
- [ ] 输入超出范围的整数时显示错误提示

### AC 8: 编辑注释
- [ ] 接受任何文本输入
- [ ] 允许空值
